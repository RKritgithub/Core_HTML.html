This is the ordered list of html tags

  1. HTML : The root element of an HTML document.
  2. HEAD : Contains meta-information about the document, such as title and linked stylesheets.
  3. TITLE : Sets the title of the web page displayed in the browser's title bar or tab.
  4. LINK : Establishes relationships between the document and external resources like stylesheets.
  5. STYLE : Contains CSS rules for styling HTML elements within the document.
  6. SCRIPT : Embeds or references client-side JavaScript code.
  7. BODY : Contains the visible content of the web page, including text, images, and other elements.
  8. H1 to H6 : Headings of varying levels, with H1 being the highest and H6 the lowest.
  9. P : Represents a paragraph of text.
  10. img : Embeds images in the document.
