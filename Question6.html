1. HTML
    Primary Use Cases Key Features/Benefits
    HTML is the backbone of web development and is used for creating the structure and content of web pages.
    • Provides the basic structure of a web page using elements like headings, paragraphs, lists, and links.
    • Allows embedding multimedia elements like images and videos.
    • Supports the creation of forms for user input.

  2. CSS
    Primary Use Cases Key Features/Benefits
    CSS is used for styling and layout control of web pages, enhancing the visual presentation.
    • Separates content from presentation, making it easier to maintain and update websites.
    • Offers a wide range of styling options, including fonts, colors, borders, and backgrounds.
    • Supports responsive design for different screen sizes and devices.

  3. JAVASCRIPT
    Primary Use Cases Key Features/Benefits
    JavaScript is a versatile scripting language used for adding interactivity and dynamic behavior to web pages.
    • Allows for real-time interactions such as form validation, image sliders, and interactive maps.
    • Supports the manipulation of the Document Object Model (DOM) for dynamically updating page content.
    • Can be used on both the client-side (in browsers) and server-side (Node.js) for full-stack development.
